# Identity REST API

This page provides a list of all identity REST API endpoints within AppsDock OS.

The guideline for the REST API can be found [here](../../../gettingstarted/guidelines/rest-api).

## The user resource

### Attributes

**id** `string`

**username** `string`

**email** `string`

**name** `string`

**status** `string`

**verified** `bool`

**penultimateLoginAt** `int`

**createdAt** `int`
